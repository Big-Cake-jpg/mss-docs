# 指令列表

本页面包含 MSS 生存服目前常用的指令列表供参考。如有需要，可以自行使用。

```
/tps 服务器卡顿情况
/server 切换服务器
    目前子服务器有：
      5th:五周目
/zm 放音乐（需要客户端安装 AllMusic 模组）
    使用 /zm play 163 歌名 来用网易云听音乐
    使用 /zm search 163 歌名 来用网易云搜索音乐(适合搜索不知道完整歌名或有多个版本的音乐)
    其他不常用指令请自行探索
/tell 游戏内私信
/tpsbar 开启/关闭服务器卡顿情况在物品栏上方/Boss 血条显示
/doll 召唤假人
    使用 /doll create <Name> 创建一个假人
    使用 /doll spawn <Name> 将假人加入服务器
    详情见：https://github.com/sjavi4/PlayerDoll/blob/main/README.md#commands
```
