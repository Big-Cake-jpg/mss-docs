# 服务器管理员守则

## 守则

:::danger
如果你已经成为了 MSS 的一位服务器管理员，请务必遵守以下规定。

如果你违反了含有“不得”一词的任意一项，你的管理员权限将被取消，你将受到严重惩罚！
:::

1. 管理员不得私自给予其他玩家管理员 / 服务器后台权限；

2. 管理员不得在服务器内整烂活 / 整人；

3. 管理员不得为其他玩家提供任何形式的便利；

4. 管理员在玩家之间遇到冲突时，要以理服人、公平公正，禁止一切形式的便利。
